Login - TorTul Storage

Pour vous connecter à TorTul Storage, veuillez fournir un webhook Discord. Il n'y a pas besoin de compte, juste ce webhook pour vous identifier.

Comment obtenir un webhook Discord :

  1. Ouvrez Discord et allez dans le serveur où vous souhaitez créer le webhook.
  2. Accédez aux paramètres du serveur en cliquant sur le nom du serveur en haut à gauche.
  3. Sélectionnez "Intégrations" puis cliquez sur "Webhooks".
  4. Créez un nouveau webhook, donnez-lui un nom et sélectionnez le canal où il doit envoyer les messages.
  5. Cliquez sur "Copier l'URL du Webhook" pour obtenir l'URL.

Collez cette URL dans le champ ci-dessus pour vous connecter.
